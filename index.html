Get started in 5 min
Just Plain Java or Kotlin

No need for XML, fragile scripting, or DSLs.
JeKa allows you to automate your entire workflow, from development to delivery, including build and pipeline using plain Java or Kotlin classes that can be easily run and debugged in an IDE..

Rich Features

In addition to its scripting capabilities and utilities, JeKa also features a complete build tool with support for compile/test/package, dependency management, Maven POM/BOM/repositories, IntelliJ/Eclipse, ` Git, Node.js, and more.

Designed for Extension

JeKa is built on a simple yet powerful plugin architecture, but it doesn't require writing specific plugins to integrate third-party technologies.
JeKa can seamlessly integrate with any tool that provides Java libraries or a command-line interface.

User Friendly

JeKa's design emphasizes simplicity, ease of use, and transparency. Its lightweight and straightforward approach, along with its simple concepts, make it easy to discover and troubleshoot.
Additionally, JeKa heavily relies on conventions to minimize the need for configuration.

Documentation

Tutorials, reference guides, and examples are available and kept in sync with JeKa's evolution. JeKa also provides rich command-line help based on class introspection, making it easy to discover and use its features
